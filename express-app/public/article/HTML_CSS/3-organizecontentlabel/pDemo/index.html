2、首字下沉
  法一：标签分隔法
    使用span将首字隔离，然后单独设置首字的样式：span{font-size:2em; float:left;}，效果如效果二
  法二：使用伪元素选择器
    伪元素first-letter指代首字，类似的伪元素还有first-line，直接设置伪元素first-letter的样式即可：.p2::first-letter{font-size:2em; float:left;}，效果如效果三
