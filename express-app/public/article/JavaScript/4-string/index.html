4、字符串常用方法

4.1、charAt() 获取相应位置的单个字符（参数：字符索引），如果参数中的索引不在0和length-1之间，则返回空字符串。 var str="hello world"; console.log(str.charAt(1));//e console.log(typeof str.charAt(-1));//string 4.2、charCodeAt() 获取相应位置的字符的Unicode编码（参数：字符索引），如果参数中的索引不在0和length-1之间，则返回NaN。 var str="hello world"; console.log(str.charAt(1));//101 console.log(str.charAt(-1));//NaN 求字符串的真实长度？ function getStrLength(str) { // 1.定义变量 var len = 0, code = 0; // 2. 遍历 for (var i = 0; i < str.length; i++) { // 2.1 Unicode码 code = str.charCodeAt(i); // 2.2 判断 if (code >= 0 && code <= 127) { len += 1; //ASCII码字符占1个字符长度 } else { len += 2; //其他字符占2个字符长度 } } // 3. 返回长度 return len; } 4.3、URI编码函数encodeURIComponent(str) 4.4、URI解码函数decodeURIComponent(str) 4.5、大小写转换 var str = "Hello World!"; console.log(str.toUpperCase()); //"HELLO WORLD!" console.log(str.toLowerCase()); //"hello world!" 4.6、应用：上传图片的格式验证
