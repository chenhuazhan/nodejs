Javascript中的对象和面向对象

1、直接量对象

JS中的直接量对象使用一对大括号来表示，大括号中包含一组键值对； JS中的对象实际上就是键值对的集合。其值可以是数据和函数，分别对应对象的属性和方法。 var person={ //属性 name:"chz", age:22, //方法 say:function(){alert("hello!");} }; person.name = "张三"; //修改属性 console.log(person.name); //访问属性 person.say(); //访问方法 使用for...in遍历对象 for(var key in person){ console.log(key+":"+person[key]); }

2、构造函数产生对象

2.1、使用Object构造 // 定义构造函数，最原始的方法，该构造函数没有参数 function Person() { var obj = new Object(); // 属性 obj.name = null; obj.age = null; // 方法 obj.study = function () { console.log(this.name + "在学习"); //使用this访问当前对象的属性 }; return obj; //返回对象 } // 使用构造函数产生对象，注意此处没有使用new var person = Person(); 2.2、使用this构造 // 构造函数，对象属性作为参数传入 function Dog(name, age) { this.name = name; this.age = age; this.eat = function (someThing) { console.log(this.name + "在吃" + someThing); }; } // 使用构造函数产生对象，注意此处需要使用new，传参个数可少于构造函数的参数 var smallDog = new Dog("小花"); console.log(smallDog); smallDog.eat("骨头"); var bigDog = new Dog("大花", 10); console.log(bigDog === smallDog); 2.3、使用this的构造函数产生对象的改进 // 构造函数,其中option是一个直接量对象 function Dog(option) { this.name = option.name; this.age = option.age; this.eat = function (someThing) { console.log(this.name + "在吃" + someThing); }; } var smallDog = new Dog({name:"小花", age:1});

3、对象的属性和方法

3.1、对象的方法也是一种属性值为函数的特殊属性 3.2、访问属性的两种方式：点操作符(obj.attr)和中括号+引号操作符(obj['attr']) 不能使用点操作符的两种情况： a.属性名中包含空格、'-'等特殊字符时 b.属性名不确定时，例如将属性名存储在一个变量里、将属性名作为函数参数 3.3、对象产生后，可以任意修改或添加其属性和方法 var obj = { name: "张三", eat: function(){alert("吃");} }; obj.age = 19; //添加一个age属性并赋值为19 obj.eat = function(){alert("喝");}; //修改对象的eat方法 obj.run = function(){alert("跑");}; //添加一个run方法 3.4、如果要访问对象中没有的属性或方法，则返回undefined 如果调用一个未定义的方法，则会报xxx is not a function错误
