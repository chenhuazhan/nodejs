ES6新增数据类型

1、set类型

1.1、数据集合，类似数组，集合中的元素是唯一的，开发中用于去除重复数据 1.2、创建集合： let s1 = new Set() //空集合 let s2 = new Set([1,2,3,3]) //构造函数传人数组，若数组中有重复元素，只取一次 1.3、获取集合大小：使用size只读属性，直接修改size值没反应 1.4、常用方法： 1.4.1、增加元素： let s = new Set([1,2,3]) s.add(4).add("hello") //支持链式调用，改变原有对象并返回 1.4.2、删除元素： s.delete(3) //返回true/false 只能传一个参数，如果传多个，只会删除第一个 1.4.3、判断元素存在性： console.log(s.has(1)) //返回true/false 1.4.4、清空元素： console.log(s.clear()) //undefined 没有返回值 1.4.5、访问key和value，返回一个构造器Iterator console.log(s.keys()) console.log(s.values()) 其s.keys()和s.values()中的内容是一样的,下面的遍历方法也证明了此事 1.5、遍历 s.forEach(function(v,i){ console.log(v,i) })

2、map类型

2.1、数据映射，类似对象 但对象的key只能是String类型，而map类型的key可以是任意类型 2.2、创建map: let m=new Map() console.log(m) //Map(0) {} m=new Map([ ['name','chz'], ['age',22], [{},'我的key是一个对象'] ]) console.log(m) //Map(3) {"name" => "chz", "age" => 22, {…} => "我的key是一个对象"} 2.3、获取大小：使用size只读属性，直接修改size值没反应 2.4、常用方法： 2.4.1、增加元素： m=new Map([ ['name','chz'], ['age',22], [{},'我的key是一个对象'] ]) m.set(function(){},"set").set([],"chz") console.log(m) //支持链式调用，改变原有对象并返回 2.4.2、获取元素值：m.get('name') 2.4.3、删除元素： console.log(m.delete({})) //false console.log(m.delete('age')) //true 2.4.4、判断元素存在性： console.log(m.has('name')) //返回true/false 2.4.5、清空元素： console.log(m.clear()) //undefined 没有返回值 2.4.6、访问key和value或两者同时访问，返回一个构造器Iterator console.log(m.keys()) console.log(m.values()) console.log(m.entries()) 2.5、遍历 m.forEach(function(v,i){ console.log(v,i) })

3、Symbol类型

3.1、Symbol类型用于声明一个独一无二的变量名，即使变量值相等，变量名也不等 3.2、声明:let str1=Symbol() let str2=Symbol('name') let str3=Symbol('name') console.log(str2===str3) //false console.log(typeof str3) //Symbol(name) 3.3、应用场景： 对象属性名同名时使用Symbol类型使得同名属性不再冲突 let obj{} obj[Symbol('name')]='chz' obj[Symbol('name')]='cjm' 这样做可保证'chz'、'cjm'两个属性值同时存在
